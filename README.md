# Smart-Factory

### <소개>
공정에서 나온 물건을 라인트레이서로 창고에 옮기고 실시간으로 창고상태, 라인트레이서 상태 등을 GUI로 확인.
3명이 팀으로 현재 역할은 C#으로 GUI구현, DB와 연동하는 것이며 프로젝트 진행 중이다. (2018-12-19)

### <제작 기간>
2018-12-01 ~ 2018-12-28
